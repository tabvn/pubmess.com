# pubmes.com

See Demo at: https://pubmess.com

Video: https://www.youtube.com/watch?v=lZZnX-8StXc
## Start Api for development

```
cd api
npm install
npm run dev

```

## Start Web App

```
cd app

npm install

npm start

```


## IOS App

```
cd IOS/PubMess
pod install
open PubMess.xcworkspace
```
